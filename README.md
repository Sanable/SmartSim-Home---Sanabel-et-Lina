# SmartSim Home - Simulation de maison intelligente

Un simulateur d'assistant vocal domotique, capable de modifier la température, la luminosité, d'ouvrir ou de fermer des volets et d'autres fonctionnalités encore.

## Prérequis
Python 3.11 nécessaire
